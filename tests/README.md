# Директория тестов

Эта директория содержит комплексные тесты для приложения SSN Search Engine.

## Файлы тестов

- **test_search_engine.py** - Комплексные тесты для класса SearchEngine с 47 тестовыми случаями, охватывающими:
  - `TestSearchEngineSetup` - Тесты инфраструктуры (5 тестов)
  - `TestSearchBySSN` - Функциональность поиска по SSN (10 тестов)
  - `TestSearchByEmail` - Функциональность поиска по Email (10 тестов)
  - `TestSearchByNameZip` - Функциональность поиска по Имени + ZIP (10 тестов)
  - `TestSearchByNameState` - Функциональность поиска по Имени + Штату (12 тестов)

## Запуск тестов

### Запуск всех тестов:
```bash
# Из корневой директории проекта
python3 -m unittest tests.test_search_engine -v

# Или используя автоматическое обнаружение тестов
python3 -m unittest discover -s tests -p "test_*.py" -v
```

### Запуск конкретного тестового класса:
```bash
python3 -m unittest tests.test_search_engine.TestSearchBySSN -v
```

### Запуск конкретного тестового метода:
```bash
python3 -m unittest tests.test_search_engine.TestSearchBySSN.test_search_by_ssn_valid_with_dashes -v
```

### Быстрый запуск тестов (только итоги):
```bash
python3 -m unittest discover -s tests
```

## Покрытие тестами

Набор тестов обеспечивает комплексное покрытие:
- ✓ Сценарии валидного ввода
- ✓ Обработка невалидного ввода
- ✓ Граничные случаи (пустые строки, null значения, специальные символы)
- ✓ Нормализация данных (форматирование SSN, регистр email, uppercase для штатов)
- ✓ Валидация ввода
- ✓ Кросс-табличные UNION запросы
- ✓ Параметры ограничения (limit)
- ✓ Форматирование JSON ответов
- ✓ Инфраструктура базы данных

## Тестовая инфраструктура

Тесты используют:
- **Временная SQLite база данных** - Изолированное тестовое окружение
- **TestSearchEngineBase** - Базовый класс с вспомогательными методами
- **setUp/tearDown** - Автоматическое заполнение и очистка тестовых данных
- **setUpClass/tearDownClass** - Инициализация и очистка базы данных

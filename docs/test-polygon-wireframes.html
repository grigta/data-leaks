Diagnosis
Запись 345-67-8901 существует в БД, но middle name в SearchBug = L, а в БД = A. Также DOB year отличается: SearchBug = 1985, БД = 1982. Ни один из 16 методов не может совпасть из-за этих расхождений.
UX notes: Этот диалог полностью повторяет Debug Flow, но встроен в контекст теста. Ключевое отличие — колонка "Why no match?" для каждого кандидата, которая объясняет почему конкретный кандидат не прошёл L2 matching. Внизу — автоматический "Diagnosis" с анализом причины неудачи.

Навигация: Кнопки ← Prev / Next → позволяют листать записи без закрытия диалога. Особенно полезно для просмотра всех NOT FOUND записей подряд.
